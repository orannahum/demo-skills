# PDF Report

Use this skill to render a PDF report.

## Steps

1. Run `scripts/generate.py` with the input data file.
2. Use the HTML template in `assets/template.html`.
3. Follow the formatting rules in `references/style-guide.md`.

Everything this skill needs lives inside its own directory.
